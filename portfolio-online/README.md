# Portfólio Online — Consultor de E-commerce

Este é um portfólio **do zero**, separado do app `desafio-30-dias-app`, criado para ser publicado como uma one page interativa.

## Como visualizar agora

> Importante: se o navegador mostrar `ERR_CONNECTION_REFUSED`, o servidor local não está rodando. Execute um comando abaixo e mantenha o terminal aberto.

Na raiz do repositório, execute:

```bash
python3 -m http.server 4173
```

Depois abra no navegador:

```text
http://127.0.0.1:4173/
```

Ou sirva diretamente esta pasta:

```bash
python3 -m http.server 4173 --directory portfolio-online
```

Depois abra:

```text
http://127.0.0.1:4173/
```

## O que a página inclui

- Hero com posicionamento profissional.
- Métricas animadas para destacar resultados.
- Cases interativos por categoria: conta milionária, do zero, 100k+/mês e prejuízo para lucro.
- Método de consultoria em quatro etapas.
- Seção de stack técnico para destacar experiência com ads, analytics, dashboards, automações e desenvolvimento.
- Formulário visual pronto para conectar a WhatsApp, CRM ou e-mail.
- Tema claro/escuro, canvas animado e layout responsivo.
